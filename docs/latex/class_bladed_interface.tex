\doxysubsection{Detailed Description}
This class provides an interface with the Bladed-\/style D\+LL. The interface was built by referencing the Bladed-\/style D\+LL of N\+R\+EL\textquotesingle{}s O\+C3 5 MW turbine. 

\begin{DoxyAuthor}{Author}
Dustin Condon 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
August 2020
\end{DoxyDate}
The interface allows the user of the class to pass the needed values to the D\+LL, and recieve the results. Inside each D\+LL there is a pitch controller, generator controller, and a low-\/pass filter on the generator shaft speed.

Note that (as far as I have seen), the Bladed-\/style D\+LL are written to be loosely coupled with the turbine model; therefore, this interface assumes that. It may be possible to write a tightly coupled controller D\+LL, however -\/ in which case this interface should be built upon to allow for it.
